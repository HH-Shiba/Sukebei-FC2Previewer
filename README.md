Sukebei FC2 Previewer is a Chrome extension, a browser add-on that enhances the browsing experience on sukebei.nyaa.si by automatically extracting and displaying FC2 video information in an injected sidebar interface.

The extension operates exclusively on sukebei.nyaa.si pages, identifying FC2-PPV video references and fetching supplementary metadata from external sources including fc2ppvdb.com and adult.contents.fc2.com. For detailed architectural information, see Architecture. For implementation specifics of individual components, see Content Script and Background Service Worker.

Purpose and Core Functionality
The FC2 Previewer extension serves as an information aggregation tool that automatically identifies FC2-PPV video references on sukebei.nyaa.si pages and provides users with comprehensive preview data through a persistent sidebar interface. The extension extracts video numbers using pattern matching, fetches preview images from multiple sources with intelligent fallback mechanisms, and displays related actress information and video recommendations.
